# Webpack4 + Vue2 + Koa2 构建SSR应用
> 
安装npm依赖
```npm
npm install
```
开发服务
```npm
npm run dev
```
生产服务, 默认会执行打包命令
```npm
npm start
```











### Configuration tasks/命令
* npm run dev : 启动开发服务
* npm start : 启动生产服务
* npm run build : 打包代码命令
* npm run build:client : 打包client
* npm run build:server : 打包server
